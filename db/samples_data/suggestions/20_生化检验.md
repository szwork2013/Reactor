## 转氨酶

###### 乙肝表面抗原阳性 + 转氨酶异常 -> 乙型肝炎待查
###### 丙肝抗体阳性 + 转氨酶异常　－＞　丙型肝炎待查

###### `升高、严重升高 与 白蛋白偏低`
###### `谷丙转氨酶或者谷草转氨酶异常，伴有白蛋白偏低，建议专科复查。`
###### 含有脂肪肝，则转氨酶不出建议？

+ 转氨酶升高（谷丙转氨酶严重升高、谷草转氨酶严重升高、γ-谷氨酰转肽酶严重升高），可伴转氨酶偏高［谷丙转氨酶偏高、谷草转氨酶偏高、γ-谷氨酰转肽酶偏高］，可伴白蛋白偏低 -> 转氨酶严重升高<肝功全项、腹部彩超>

+ 转氨酶偏高（谷丙转氨酶偏高、谷草转氨酶偏高、γ-谷氨酰转肽酶偏高），可伴白蛋白偏低 -> 转氨酶偏高<肝功全项、腹部彩超>

1.忌辛辣、戒烟禁酒、忌生活不规律；2.忌食加工食品、乱用补品；3.忌蛋白饮食、高铜饮、情志不畅。

## 血清蛋白

+ 总蛋白偏低，白蛋白偏低

建议专科治疗。

+ 总蛋白偏高，白蛋白偏高

注意饮食，建议专科治疗。

+ 总蛋白偏低，白蛋白偏高

肝功能异常，建议专科治疗。

+ 总蛋白偏高，白蛋白偏低

建议专科进一步检查。

+ 总蛋白偏低，球蛋白偏低

总蛋白低是因为白蛋白低造成的，专科医院检查乙肝五项和丙肝，排除病毒性肝炎导致的白蛋白生成减少；定期复查。

+ 总蛋白偏低

见于水钠潴留致血浆稀释、营养不良、吸收不良、消耗性疾病（严重结核、甲状腺功能亢进、恶性肿瘤）、合成障碍（肝功能受损）、蛋白质丢失增多等。建议：结合临床进一步检查。

+ 总蛋白偏高

见于脱水（呕吐、腹泻、高热、休克）、外伤性休克、慢性肾上腺皮质功能不全、多发性骨髓瘤建议：结合临床进一步检查。

+ 白蛋白偏低

见于水钠潴留致血浆稀释、营养不良、吸收不良、消耗性疾病（严重结核、甲状腺功能亢进、恶性肿瘤）、合成障碍（肝功能受损）、蛋白质丢失增多等。建议:结合临床进一步检查。

+ 白蛋白偏高

见于脱水（呕吐、腹泻、高热、休克）、外伤性休克、慢性肾上腺皮质功能不全、多发性骨髓瘤建议:结合临床进一步检查。

+ 球蛋白偏低

见于生理性减少、免疫功能抑制、低γ球蛋白血症等。建议：结合临床进一步检查。

## 胆红素、胆汁酸

+ 胆红素异常（总胆红素偏高、直接胆红素偏高、间接胆红素偏高）<肝功全项、腹部彩超>

1. 内科复查，明确诊断；2. 养成良好的生活习惯，清淡饮食；3. 定期复查肝功能、腹部超声。

+ 碱性磷酸酶偏高<肝功全项、腹部彩超>

碱性磷酸酶异常多见于阻塞性黄疸，原发、继发性肝癌，胆汁淤积性肝炎等。建议：内科复查，明确诊断；

+ 总胆汁酸偏高<肝功全项、腹部彩超>

见于急慢性病毒性肝炎、胆汁瘀滞、慢性乙醇中毒、肝硬化、原发性肝癌、胆道梗塞等。建议：结合临床进一步检查。

## 胆碱酯酶
####### _如何结合超声科肝脏常见疾病 与 检验科肝脏相关异常_

+ 胆碱酯酶偏高

见于神经系统疾病、甲状腺功能亢进、糖尿病、高血压、支气管哮喘、Ⅳ型高脂蛋白血症、肾功能衰竭等。建议内科复查，明确诊断。

+ 胆碱酯酶偏低

可见于有机磷中毒、肝炎、肝硬化、营养不良、恶性贫血、急性感染、心肌梗死、肺梗死、肌肉损伤、慢性肾炎、皮炎及妊娠晚期建议内科复查，明确诊断。

+ 同型半胱氨酸偏高

可致心血管疾病、神经精神疾病、加速衰老、免疫力降低等。建议内科复查，明确诊断；补充叶酸、维生素B6、维生素B12；低脂饮食，多食水果、蔬菜。

## 肾功

+ 肌酐偏高，可伴异常［尿素氮偏高、高尿酸血症］<尿常规、腹部彩超、肾功全项>

见于各种肾病、急性或慢性肾功能衰竭等。建议：结合临床，肾内科进一步检查。

+ 肾功能严重异常（肌酐严重升高、尿素氮升高）<尿常规、腹部彩超、肾功全项>

及时肾内科就诊。

+ 尿素氮偏高，可伴高尿酸血症<尿常规、腹部彩超、肾功全项>

清淡饮食，一周后复查肾功能，如仍异常请肾内科就诊。

+ 尿素氮升高，高尿酸血症<尿常规、腹部彩超、肾功全项>

肾内科就诊。

+ 高尿酸血症<尿常规、腹部彩超、肾功全项>

1.限制含高嘌呤食物（如豆制品、海鲜、菌类、动物内脏等）摄入；2.戒酒；3.多饮水；4.必要时内分泌科就诊。

+ 尿酸偏低<尿常规、腹部彩超、肾功全项>

1.调整饮食结构；2.复查肾功能，必要时肾内科就诊。

+ 血脂异常（胆固醇偏高、甘油三酯偏高、低密度脂蛋白偏高、高密度脂蛋白偏低），可伴脂蛋白异常［脂蛋白a偏高、载脂蛋白A1偏高］<血脂全项>

1.改变不合理生活方式（戒烟限酒、每人每天摄入盐量小于6g）；2.调整饮食结构（多食全谷食物、水果、蔬菜，少食饱和脂肪酸、胆固醇）；3.控制体重在标准范围内；4.增加体育锻炼，每周5-7天保持每天30-60分钟运动时间（如快走、慢跑、游泳、太极拳等）；5.监测血压，控制在正常范围内。6-8周后复查血脂，如未达标，心内科就诊。

## Part 4: 糖尿病史与血糖

###### 有糖尿病史，则一元化诊断为糖尿病

### 生化 与 尿酮体

+ 糖尿病史？，血糖控制不理想（空腹血糖偏高？、空腹血糖高于诊断标准？、糖化血红蛋白偏高？、糖化血红蛋白高于诊断标准？），尿常规酮体阳性 -> 糖尿病，酮症酸中毒待查<糖尿病检测组合>

尽快到内分泌科诊治。

+ 糖尿病史，血糖控制不理想（空腹血糖偏高、空腹血糖高于诊断标准、糖化血红蛋白偏高、糖化血红蛋白高于诊断标准），可伴尿葡萄糖阳性 -> 糖尿病，血糖控制不理想<糖尿病检测组合>

内分泌科就诊。

+ 糖尿病史 -> 糖尿病<糖尿病检测组合>

1.合理控制总热量及食物成分比例，适当增加食物纤维摄入量，减少饱和脂肪酸摄入；2.根据身体情况，规律的进行体育运动；3.按时进行药物治疗；4.加强自我血糖监测，血糖控制不理想时，及时到内分泌科就诊。

+ 尿葡萄糖阳性<糖尿病检测组合>

见于糖尿病、甲状腺功能亢进、妊娠后期等。摄入强化剂药物可出现假阳性。服用维生素C超过500mg/L，可出现假阴性。建议结合临床进一步检查。

+ 血糖高于诊断标准（空腹血糖高于诊断标准、糖化血红蛋白高于诊断标准），可伴血糖高［空腹血糖偏高、糖化血红蛋白偏高］ -> 糖尿病待查<糖尿病检测组合>

内分泌科就诊，明确诊断及治疗。

+ 血糖高（空腹血糖偏高、糖化血红蛋白偏高）<糖尿病检测组合>

近期复查空腹血糖；如血糖仍偏高，内分泌科就诊，明确是否是糖尿病前期。

##### _建议：对于高危客户，出去筛查空腹血糖以外，加查糖化血红蛋白。推送至销售，院长。_

## Part 5: 心肌酶

###### _确定心肌酶异常提示方法？_

+ 心肌酶异常（乳酸脱氢酶偏高、α-羟基丁酸脱氢酶偏高、肌酸激酶偏高、肌酸激酶同工酶偏高、乳酸脱氢酶升高、α-羟基丁酸脱氢酶升高、肌酸激酶升高、肌酸激酶同工酶升高）

心肌酶升高常由于心肌损伤引起。常见于心肌炎、急性心梗等，但有些肌肉疾病也可以导致心肌酶升高，建议：内科复查，结合临床症状明确诊断。

## Part 6: RF、ASO、CRP
###### _有风湿病史，RF、ASO、CRP异常的结合怎么建议_

+ 类风湿因子偏高，抗“o”偏高，C-反应蛋白偏高
+ 类风湿因子偏高，抗“o”偏高

类风湿因子（RF）、抗“o”（ASO）同时升高多见于风湿的急性活动期建议内科复查，明确诊断。

+ 类风湿因子偏高，C-反应蛋白偏高
+ 类风湿因子偏高

类风湿因子（RF）在类风湿性关节炎患者中的检出率很高；血管炎、肝病、慢性感染时亦可出现建议内科复查，明确诊断。

+ 抗“o”偏高，C-反应蛋白偏高
+ 抗“o”偏高

抗“o”（ASO）升高见于上呼吸道、皮肤或软组织感染，A群溶血性链球菌所致的败血症以及类风湿性关节炎等。建议内科复查，明确诊断。

##### _C-反应蛋白对风湿没有特异性_

+ C-反应蛋白偏高

C-反应蛋白（CRP）升高见于各种急性炎症、组织损伤、心肌梗塞、手术创伤、放射性损伤等疾病建议内科复查，明确诊断。
